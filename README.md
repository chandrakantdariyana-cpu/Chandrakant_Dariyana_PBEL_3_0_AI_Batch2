# AI-Based Cyber Threat Detection Framework

## 📌 Project Overview

This project is an AI-based Cyber Threat Detection Framework that detects malicious network traffic using Machine Learning techniques on the CICIDS2017 dataset.

## 🚀 Features

- Dataset Merging
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Selection
- Machine Learning Model Training
- Attack Prediction
- Streamlit Dashboard (Coming Soon)
